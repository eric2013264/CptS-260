This program outputs the number in ascii code.

Thus the sample input/output 5+7=12 outputs l, the ascii value of 108, sum of ascii values of 5 and 7
—————————————
Please remember to restart the program if the output is incorrect.
Ready bits can be easily messed up if tasks aren’t done in the correct order.

Steps to ensure proper output:

1. open the .asm file
2. go to Run > Assemble
3. go to Tools > Keyboard and Display MMIO Simulator
4. hit connect to MIPS
4. go to Run > Go
6. type the equation in the form x+y= or x-y=
